--------------------------------------MinStack.java--------------------------------------

The MinStack.java file contains two classes, MinStack and Min. MinStack is an array-based
Stack class that supports push, pop, top, and retrieving the minimum element 
(by utilizing the Min class). To test this program, go to the main and initialize a new 
MinStack object with a maxsize (will change dynamically and always be nonzero regardless 
of integer input parameter). After that, use the objectname. push(x), pop(), top(), or
getMin() to test various scenarios.

--------------------------------------EvalExp.java--------------------------------------

The EvalExp.java file contains a Stack class, and three additional functions. To test an
input, simply run the program and input an expression with a constant -- operant -- 
expression format. For example, 4<3*4-9+1 is a valid input, while 99-4=3 is not valid.
